"Perangkat storage Dell EMC Data Domain dan Integrated Data Protection Appliance (IDPA) mendapat pembaruan. Pembaruan tersebut diklaim telah dirancang untuk menawarkan fleksibilitas dan kemampuan multi-cloud yang lebih luas. President Data Protection Dell EMC Beth Phalen mengatakan, Dell EMC berkomitmen menghadirkan inovasi berkelanjutan di portofolio perlindungan data yang mendukung dan meningkatkan adopsi lingkungan multi-cloud bagi pelanggan. “Perangkat data kami sangat canggih, mudah dikelola dan mudah dikembangkan ke cloud publik dengan kemampuan cloud asli,"" katanya dalam keterangan resminya kepada JawaPos.com. Dalam kesempatan yang sama, Dell EMC juga menggandeng Alibaba Cloud dan Google Cloud Platform sebagai layanan baru cloud publik mereka. ""Keduanya melengkapi penawaran Cloud Tier Dell EMC kepada para pelanggan yang ingin memanfaatkan cloud publik untuk retensi jangka panjang,” terang Alex Lei, VP Sales, Data Protection Solutions, Asia Pasifik dan Jepang, Dell EMC. Mendukung peningkatan performa multi-cloud, Dell EMC didukung perangkat lunak Data Domain OS 6.2 dan IDPA 2.3. Dengan hal tersebut, mereka mengklaim pelanggan kini memiliki lebih banyak pilihan untuk memperluas perlindungan data ke cloud publik dengan layanan Cloud Tier yang diperluas mencakup Google Cloud Platform dan Alibaba Cloud yang sudah tersedia di Indonesia. Kemampuan ini merupakan fitur tambahan dari layanan yang telah ditawarkan di Amazon Web Service (AWS), Microsoft Azure, Dell EMC Elastic Cloud Storage, Virtustream, Ceph, IBM Cloud Open Storage, AWS Infrequent Access, Azure Cool Blob storage, dan Azure Government Cloud, di negara-negara tertentu. Selain itu, Free-space Estimator Tool baru untuk Cloud Tier juga diklaim membantu memungkinkan pengelolaan kapasitas yang lebih efisien untuk membantu mengurangi biaya storage on-premise dan cloud. Dell EMC juga memperluas ekosistem penyedia cloud publik untuk Data Domain Virtual Edition (DD VE). Ini menyediakan perlindungan data berbasis perangkat lunak on-premise dan di cloud publik yang mencakup AWS GovCloud, Azure Government Cloud, dan Google Cloud Platform. Ekosistem cloud yang diperluas dan dikombinasikan dengan peningkatan kapasitas DD VE telah diumumkan sebelumnya hingga 96 TB. Hal ini memastikan lingkungan cloud pelanggan yang terus berkembang akan menerima tingkat perlindungan yang sama seperti diterima dari perangkat Dell EMC on-premise. Selain itu, Native Cloud Disaster Recovery kini telah tersedia di seluruh portofolio IDPA. Hal tersebut memungkinkan pelanggan untuk melakukan failover yang hemat biaya ke lingkungan cloud secara end-to-end. Pelanggan tidak lagi harus mengeluarkan biaya serta mendirikan dan mengelola lokasi kedua untuk disaster recovery. Mereka bisa melakukan failover ke cloud publik dengan mudah jika terjadi masalah dan melakukan failback ketika semua masalah sudah diatasi. Sistem Operasi Dell EMC Data Domain 6.2, Dell EMC IDPA 2.3, dan pembaharuan untuk Data Domain DD3300 telah tersedia di Indonesia melalui mitra channel resmi Dell EMC."
